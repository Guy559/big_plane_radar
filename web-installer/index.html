Big Plane Radar

Browser installer for the Waveshare ESP32-S3-Touch-LCD-7 and LCD-7B. Use a Chromium-based browser with Web Serial support. Set the board switch to UART1 and plug USB into the UART1 port before connecting.

Web Serial is not available. Use Chrome, Edge, or another Chromium-based desktop browser over HTTPS.

Select the board you actually have before installing — the two images aren't interchangeable (different resolution and IO expander driver). Flashing the wrong one won't damage the hardware, but the display/touch won't work correctly until you flash the matching image.
